# /ai-llm-suite

Build production AI & LLM applications with expert guidance on RAG systems, AI agents, ML ops, and prompt engineering.

## Usage

```
/ai-llm-suite [subcommand]
```

## Subcommands

### `rag` — Design or Debug a RAG System

Design RAG architectures, debug retrieval quality, optimize chunking and embedding strategies.

```
/ai-llm-suite rag
```

### `agent` — Build an AI Agent

Design agent architectures (ReAct, plan-and-execute, reflection), implement tool use, memory systems, and multi-agent orchestration.

```
/ai-llm-suite agent
```

### `deploy` — Deploy and Optimize LLM Infrastructure

Model serving setup (vLLM, TGI, Ollama), inference optimization, cost management, monitoring, and production patterns.

```
/ai-llm-suite deploy
```

### `prompt` — Design and Test Prompts

System prompt design, few-shot engineering, structured output, guardrails, evaluation, and red-teaming.

```
/ai-llm-suite prompt
```
